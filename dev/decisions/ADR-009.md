
# ADR-009: Política de datos para RAG y entrenamiento

## Fecha
2025-12-04

## Estado
Aceptada

## Contexto
MolSys-AI incluirá:
- un sistema RAG,
- datasets de entrenamiento LoRA especializados.

Es esencial definir qué datos pueden usarse.

## Decisión
Fuentes de datos **permitidas**:

1. Documentación pública de:
   - MolSysMT
   - MolSysViewer
   - TopoMT
2. Docstrings y código fuente **público**.
3. Ejemplos, tutoriales y notebooks públicos.
4. Issues y discusiones públicas en GitHub (curadas).

Fuentes **prohibidas**:

- Datos clínicos, personales o confidenciales.
- Logs internos con información privada.
- Conversaciones internas de laboratorio no públicas.
- Archivos que contengan rutas, claves o configuraciones sensibles.

## Opciones consideradas
- Incluir datos internos → **rechazada** por riesgos.
- Limitarse solo a docs Sphinx → demasiado restrictivo.

## Justificación
Mantener el dataset completamente reproducible y abierto evita problemas legales y facilita que otros usen el modelo localmente.

## Consecuencias
- El modelo podrá ser liberado con una licencia abierta.
- El dataset podrá ser publicado en Hugging Face datasets.

## Notas futuras
- Podrán añadirse datasets sintéticos generados por el propio modelo.
